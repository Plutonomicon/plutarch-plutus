revAppend :: forall a. [a] -> [a] -> [a] Source #

Cons each element of the first list to the second one in reverse order (i.e. the last element of the first list is the head of the result).

revAppend xs ys === reverse xs ++ ys
>>> revAppend "abc" "de"
"cbade"
